DrawUI.js Component Index

This index documents the maintained public surface of the DrawUI package. Sections follow the class hierarchy: Text and Inputs are Control leaves; Layout, Collections, Panels, Overlays, and Shell are Container branches; Peers need external packages; Facade is the package entry surface.

Import Surfaces

Use package exports for all public DrawUI consumption:

import { DrawUI, FloatingWindow, Button, RadialMenu } from "drawui";
import "drawui/styles/core.css";

Status Legend

  • Stable: documented, maintained package API.
  • Experimental: public, but behavior or styling may still evolve.
  • Internal: repository code that should not be treated as package API.

See also the API reference for per-symbol pages.

DrawUI Factory Additions

  • new Grid() — Package-owned CSS grid layout primitive.
  • new IconButton(label, options) — Tile-style action button without module-local styling.
