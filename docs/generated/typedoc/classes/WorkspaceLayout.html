    Class WorkspaceLayout

    Workspace grid manager: region open/close/size, stack mount points, and panel registry. Does not create or style panel chrome — docked UI is a WorkspacePanel (CollapsiblePanel subclass) owned by the app.

    Example: Note

    return new Disclaimer(
      "WorkspaceLayout requires a full #World DOM tree — use the templates example app.",
    );
