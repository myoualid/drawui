Chart

Status: Stable
Category: peers
Import: drawui
Emitted class(es): Rectangle, chart-ui-component

Chart.js-backed visualization panel used by DrawUI.Chart().

Import

import { Chart } from "drawui";

Examples

Bar

return makeChart({
    type: "bar",
    data: {
      labels: CHART_LABELS,
      datasets: [
        {
          label: "Tickets",
          data: [12, 19, 8, 15, 22],
          backgroundColor: CHART_PALETTE[0],
          borderColor: CHART_BORDERS[0],
          borderWidth: 1,
        },
      ],
    },
  });
