InstructionPanel

Status: Stable
Category: panels
Import: drawui
Emitted class(es): None by default

Titled panel of keyboard / pointer instruction rows.

Import

import { InstructionPanel } from "drawui";

Examples

Instruction panel

return new InstructionPanel("Viewport", "3d_rotation", [
    ["MMB", "Orbit around the scene"],
    ["Shift + MMB", "Pan the view"],
    ["Scroll", "Zoom in / out"],
    ["Alt + MMB", "Roll the camera"],
  ]);
