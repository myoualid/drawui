Markdown

Status: Stable
Category: text
Import: drawui
Emitted class(es): Markdown

Renders markdown (or sanitized HTML) into a .Markdown container. Min build uses the built-in converter. Full build exports a Showdown-backed Markdown with the same API when peers are loaded.

Source: src/components/Markdown.js

Import

import { Markdown } from "drawui";

Examples

Markdown

return new Markdown(
    [
      "## Min-build markdown",
      "",
      "Built-in converter — **no Showdown**. Supports headings, **bold**, *italic*,",
      "`inline code`, lists, and fenced blocks.",
      "",
      "- Item one",
      "- Item two",
      "",
      "1. Ordered",
      "2. Lists",
      "",
      "```js",
      "new Markdown(\"# Hello\");",
      "```",
    ].join("\n"),
  );
