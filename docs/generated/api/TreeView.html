TreeView

Status: Stable
Category: collections
Import: drawui
Emitted class(es): dui-tree, dui-tree-item, dui-tree-toggle, dui-tree-label, dui-tree-children, is-expanded, is-selected

Expandable nested tree for hierarchical data.

Shares the same options shape as NavigationList (getChildren, getLabel, onItemClick, renderItem) so the same { name, children } trees work for both.

Source: src/components/lists/TreeView.js

Import

import { TreeView } from "drawui";

Examples

TreeView

const sampleTree = {
    name: "Project",
    children: [
      {
        name: "Site",
        children: [
          { name: "Building A", children: [{ name: "Level 01" }, { name: "Level 02" }] },
          { name: "Building B" },
        ],
      },
      { name: "Library", children: [{ name: "Materials" }, { name: "Profiles" }] },
    ],
  };
  const tree = new TreeView({
    getLabel: (item) => item.name,
    getChildren: (item) => item.children || [],
    onItemClick: (item) => console.log("selected", item),
  });
  tree.setData(sampleTree);
  return tree.panel;
