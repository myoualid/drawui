DataGrid

Status: Stable
Category: peers
Import: drawui
Emitted class(es): Rectangle, ag-grid-custom

AG Grid-backed spreadsheet panel used by DrawUI.Spreadsheet().

Import

import { DataGrid } from "drawui";

Examples

DataGrid

const sheet = new DataGrid({
    height: "280px",
    data: [
      { name: "Ada Lovelace", role: "Analyst", score: 98 },
      { name: "Alan Turing", role: "Engineer", score: 95 },
      { name: "Grace Hopper", role: "Lead", score: 97 },
    ],
    columnNameMapper: {
      name: "Name",
      role: "Role",
      score: "Score",
    },
  });

  return afterAttach(sheet, () => sheet.init());
