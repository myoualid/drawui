Toast

Status: Stable
Category: panels
Import: drawui
Emitted class(es): Toast, Toast--{type}, Toast--fade-out, Toast-close, status-bar, status-bar-fill

Transient notification or progress overlay.

Notification:

new Toast("Changes saved", "success");

Progress:

const toast = new Toast();
toast.show("#host", "Exporting…");
toast.update(50, "50%");
toast.hide();

Source: src/components/feedback/Toast.js

Import

import { Toast } from "drawui";

Examples

Progress toast

const inlineHost = new Container();
  inlineHost.dom.id = "inline-progress-host";
  inlineHost.setStyle("position", ["relative"]).setStyle("minHeight", ["3rem"]);
  const toast = new Toast();

  return new StackPanel({ isVertical: true })
    .gap("0.75rem")
    .add(
      new Button("Run inline progress").addClass("secondary").onClick(() => {
        toast.show("#inline-progress-host", "Exporting…");
        let pct = 0;
        const timer = setInterval(() => {
          pct += 10;
          toast.update(pct, `${pct}%`);
          if (pct >= 100) {
            clearInterval(timer);
            setTimeout(() => toast.hide(), 400);
          }
        }, 120);
      }),
    )
    .add(inlineHost);
