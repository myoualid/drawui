Grid

Status: Stable
Category: layout
Import: drawui
Emitted class(es): Grid

Grid layout primitive with explicit column or auto-fit helpers.

Import

import { Grid } from "drawui";

Examples

Grid

const grid = new Grid();
  grid.setStyle("display", ["grid"]);
  grid.setStyle("gridTemplateColumns", ["repeat(3, 1fr)"]);
  grid.setStyle("gap", ["0.5rem"]);
  ["A", "B", "C"].forEach((label) => {
    grid.add(
      new Card()
        .add(new TextBlock(label))
        .setStyle("padding", ["0.5rem"])
        .setStyle("textAlign", ["center"]),
    );
  });
  return grid;
